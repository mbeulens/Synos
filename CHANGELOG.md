# Changelog

## v2.1.0 — Minor release

### Improvements
- **Stream management** — Clean rows with right-click context menu: Move Up, Move Down, Edit, Delete
- **YouTube/SoundCloud thumbnails** — Video thumbnails shown as album art instead of MusicBrainz lookup
- **Breadcrumb fix** — Back button navigates correctly through all Music Services submenus

### Bug Fixes
- Reverted drag & drop and double-click changes that broke panel layout

## v2.0.0 — Minor release

### YouTube Music — Fully Working
- Search songs via ytmusicapi (no login required)
- Liked Music playlist built-in
- Add playlists manually by pasting URL or ID
- Download & convert audio to MP3 via yt-dlp + ffmpeg
- Auto-skip unavailable tracks (Shorts, removed videos)

### SoundCloud — Fully Working
- Search tracks, My Tracks, My Playlists
- Direct MP3 playback with seek slider support
- Profile URL and browser cookies configurable in Settings
